AddressBookImporter
    by Albert Ramstedt, Adocca AB
    http://www.adocca.com

== DESCRIPTION:
  
A mechanize scraper for collecting email contacts.

== FEATURES/PROBLEMS:
  
Only implemented for Gmail, Hotmail and Yahoo. New hotmail live beta thing not working atm. Please insert
your own test accounts if you want to test the validity of the parser.
